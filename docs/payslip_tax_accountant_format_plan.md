# 급여명세서 세무사 형식 수정 계획

## 📋 목표
세무사가 실제 발행한 명세서와 동일한 형식으로 표시

## 📊 세무사 명세서 형식

### 지급 내역
- 기본급: 2,340,000원
- 식대: 160,000원 (별도 지급으로 명기)

### 공제 내역
- 건강보험: 82,950원
- 고용보험: 4,500원
- 장기요양보험료: 10,740원
- 공제액계: 98,190원

### 차인지급액
- 차인지급액 = 기본급 - 공제액계 = 2,241,810원
- 세금(3.3%): 77,220원 (별도 표시, 공제액계에 포함 안함)

### 식대
- 식대: 160,000원 (별도 지급)

## 🔧 수정 필요 사항

### 1. 공제 내역 표시 수정
- 국민연금 제거 (세무사 명세서에 없음)
- 건강보험, 고용보험, 장기요양보험료만 표시
- 세금은 별도 표시 (공제액계에 포함 안함)

### 2. 차인지급액 계산
- 차인지급액 = 기본급 - 공제액계 (세금 제외)
- 실수령액 = 차인지급액

### 3. 식대 표시
- 식대를 "별도 지급"으로 명기
- 지급 내역에서 식대를 별도 행으로 표시

### 4. 수정 대상 파일
1. `src/app/admin/payslip-generator/page.tsx`
   - `printSavedPayslipTaxAccountantStandard` 함수
   - `printDetailedSavedPayslip` 함수
   - `printDetailedPayslip` 함수
   - 모달 UI 표시 부분

## ✅ 검증 기준
1. 공제액계 = 건강보험 + 고용보험 + 장기요양보험료 = 98,190원
2. 차인지급액 = 기본급 - 공제액계 = 2,241,810원
3. 세금은 별도 표시 (공제액계에 포함 안함)
4. 식대는 "별도 지급"으로 명기
