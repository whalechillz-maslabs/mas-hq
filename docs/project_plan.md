## 새로 추가된 기능: 테스트 로그인

### 기능 개요
관리자가 직원 관리 페이지에서 해당 직원의 계정으로 새창에서 로그인하여 테스트할 수 있는 기능

### 주요 특징
1. **새창에서 테스트**: 직원 계정으로 별도 창에서 로그인
2. **자동 입력**: 전화번호, 비밀번호, 핀번호 자동 입력
3. **테스트 모드 표시**: 테스트 중임을 명확히 표시
4. **읽기 전용 폼**: 테스트 모드에서는 입력 필드 수정 불가
5. **보안 고려**: URL 파라미터로 정보 전달하되 안전하게 처리

### 사용 방법
1. 관리자로 로그인
2. 직원 관리 페이지 접근
3. 원하는 직원의 "테스트 로그인" 버튼 클릭 (🔗 아이콘)
4. 새창에서 해당 직원 계정으로 자동 로그인
5. 테스트 완료 후 창 닫기

### 테스트 스크립트
```bash
npm run test:admin-login
```
