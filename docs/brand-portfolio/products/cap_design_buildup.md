## 디자인 컨셉

### 프로젝트명
**MASSGOO × MUZIIK 콜라보 기념 모자**

### 디자인 목적
- 마쓰구와 뮤직의 콜라보레이션 기념
- 브랜드 아이덴티티 강화
- 프리미엄 골프 브랜드 이미지 구축

### 디자인 방향
- 심플하고 세련된 디자인
- 두 브랜드의 균형 잡힌 표현
- 골프장 환경에 적합한 고급스러움

---

## 디자인 결정 과정

### 1단계: 초기 아이디어

#### 옵션 1: 콜라보 메인 디자인
- **전면 상단**: MASSGOO 마쓰구 기호로고 + 문자로고
- **전면 하단**: MUZIIK 뮤직 기호로고 + 문자로고
- **배치**: 상하 배치

#### 옵션 2: 심플 원형 로고 디자인
- **전면 중앙**: MASSGOO 마쓰구 원형로고 (150mm)
- **전면 하단**: MUZIIK 뮤직 문자 로고

### 2단계: Muziik 가이드라인 검토

#### Muziik OEM 가이드라인 주요 사항
- **Equal Visual Weight**: 두 브랜드의 시각적 무게가 동등해야 함
- **기호로고 사용 제한**: 기호로고만 사용 시 문제 제기 가능
- **문자로고 사용 권장**: 문자로고만 사용하는 것이 안전

#### 가이드라인 준수를 위한 조정
- 기호로고 삭제하고 문자로고만 사용
- MASSGOO와 MUZIIK의 시각적 균형 유지
- "X"를 중간에 배치하여 콜라보 표현

### 3단계: 최종 디자인 확정

#### 최종 선택: 텍스트 로고 콜라보 디자인
- **전면**: MASSGOO × MUZIIK 텍스트 로고
- **로고 구성**: 
  - MASSGOO (문자로고만)
  - × (콜라보 표시)
  - MUZIIK (문자로고만)

## 최종 디자인 스펙

### 로고 배치

#### 전면 (Front)
- **레이아웃**: 중앙 정렬
- **로고 구성**:
  ```
  MASSGOO
     ×
  MUZIIK
  ```
- **로고 크기**:
  - MASSGOO: 6.5cm
  - MUZIIK: 6cm
  - X: 2cm
- **정렬**: 중앙 정렬, 세로 배치

### 색상 조합

#### 옵션 1: 다크 배경
- **배경**: 네이비, 블랙
- **텍스트**: 화이트
- **특징**: 고급스럽고 세련된 느낌

#### 옵션 2: 라이트 배경
- **배경**: 베이지, 화이트
- **텍스트**: 검정
- **특징**: 깔끔하고 모던한 느낌

#### 최종 선택
- **네이비**: 5개 (화이트 텍스트)
- **블랙**: 5개 (화이트 텍스트)
- **베이지**: 5개 (검정 텍스트)
- **화이트**: 5개 (검정 텍스트)

### X (콜라보 표시) 색상 결정
- **제주돌담체** 폰트 사용 검토
- 최종적으로 텍스트와 동일한 색상 사용
