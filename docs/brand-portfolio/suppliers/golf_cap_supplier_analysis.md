## API 연동 전략

### 마플 (MARPPLE) - 추천
1. **1차 접근**: 고객센터(help@marpple.com)에 API 연동 문의
2. **대안**: Playwright를 활용한 웹 자동화
   - 주문 프로세스가 구조화되어 있어 자동화 가능성 높음
   - 단체/대량 주문 시스템 활용

### 커스텀존 (CustomZone)
1. **1차 접근**: customzone@naver.com에 API 연동 문의
2. **대안**: 디자인 툴 API 또는 웹 자동화
   - iframe 기반 디자인 툴은 복잡할 수 있음

### 중국 업체 (Sumkcaps, TALMUD)
- API 제공 가능성 낮음
- Playwright를 활용한 문의 폼 자동화 고려
- 또는 직접 이메일/전화 협의
