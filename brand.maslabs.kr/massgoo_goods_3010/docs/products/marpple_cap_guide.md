## 로고 등록 방법

### 방법 1: 웹 에디터에서 직접 업로드
1. 상품 페이지에서 **"이미지 업로드"** 버튼 클릭
2. PNG 또는 JPG 파일 업로드
3. 이미지 크기 조정 및 위치 설정
4. 주문 진행

### 방법 2: 전문 인쇄용 파일 전송
1. 웹 에디터에서 주문 완료
2. 주문자 정보와 함께 원본 파일 전송:
   - **이메일**: help@marpple.com
   - **1:1 채팅**: 마플 웹사이트 내 채팅
3. 전송 정보:
   - 주문번호
   - 성함
   - 연락처
   - 원본 파일 (PDF, AI, PSD)

### 파일 요구사항
- **형식**: PNG, JPG (웹 업로드) / PDF, AI, PSD (전문 인쇄)
- **크기**: 최소 2,500px 이상 권장 (인쇄 사이즈 20cm 기준)
- **주의**: 작은 이미지를 확대하면 화질이 깨질 수 있음
